# Sample Benchmarks

These directories are benchmark definitions for validating the runtime once real Godot sample projects are created.

- `greybox-controller`: 2D movement, jump, and camera framing
- `ui-menu`: focus order, hover state, and menu visibility
- `cutscene-demo`: timed captures and cutscene completion

Each sample currently includes:

- scenario JSON for the CLI validator
- a short implementation brief describing what the Godot sample should contain
